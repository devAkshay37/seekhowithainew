# SeekhoWithAI Documentation

SeekhoWithAI is your intelligent teaching companion, transforming lesson topics into classroom-ready experiences.

## Project Overview

Welcome to the documentation for **SeekhoWithAI**, an AI-powered teaching companion for school teachers.

SeekhoWithAI transforms complex lesson preparation into a structured, curriculum-aligned, and efficient process. Built with teachers in mind, it supports multiple regional languages and ensures a premium user experience on both mobile and desktop.

### At a Glance:
- **Project Structure**: Next.js 15 (App Router).
- **Authentication**: Supabase Auth (Support for Google OAuth).
- **Primary AI**: Google Gemini Flash API (`gemini-1.5-flash`).
- **Database**: Supabase PostgreSQL + RLS.
- **Reporting & Analytics**: PostHog.

### Our Mission
To empower teachers by reducing time spent on administrative and preparation tasks, allowing them to focus on what matters most: **teaching and student engagement**.
